Create

Create a new package

Status: partial - location header not properly filled in

Path: /packages/

Method: PUT

Parameters:

  • name: package name
  • epoch: package epoch
  • version: package version
  • release: package release
  • arch: package architecture
  • description: package description
  • checksum_type: type of package checksum
  • checksum: package checksum
  • filename: package filename

Authentication:

username/pass

Filters:

None

Response: 201 Created

Headers:

  • location - path to package resource 
    {'arch': 'noarch',
 'checksum': {'md5': '41be1e70179ddf079ceda621df818341'},
 'description': 'test package',
 'epoch': '0',
 'filename': 'my_package.pkg',
 'id': '12d14ebe-3ea7-43d2-9119-d278d2552c09',
 'name': 'my_package',
 'provides': [],
 'release': '2',
 'requires': [],
 'version': '1'}

Error: 409 Conflict if there is a package with the same: name, version, revision, epoch, arch, and checksum
