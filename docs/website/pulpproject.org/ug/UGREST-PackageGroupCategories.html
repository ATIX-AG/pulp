Once you get a repository created you can start managing package group categories. Package group categories exist as members of a repository.

A package group category is a collection of package group ids

Base URL Path: /repositories/<repoid>/

Note: PackageGroupCategory operations are served from the repository controller

List

List the package group categories under a repository

Status: complete

Path: /repositories/<repoid>/packagegroupcategories/

Method: GET

Parameters:

None

Authentication:

username/pass or Consumer Certificate

Filters:

None

200 OK 

{"multimedia": 
  {"description": "multimedia created from package kit group data", "repo_defined": false, "display_order": 99, "immutable": false, "translated_name": {}, "packagegroupids": ["sound-and-video"], "translated_description": {}, "_id": "multimedia", "id": "multimedia", "name": "multimedia"}

Error: 404 Not Found if there is no repository

Create

Create a new packagegroup

Status: partial - location header not properly filled in

Path: /repositories/<repoid>/create_packagegroupcategory/

Method: PUT

Parameters:

  • categoryid: package group category id
  • categoryname: package group category name
  • description: package group category description

Authentication:

username/pass

Filters:

None

Response: 201 Created

Error: 409 Conflict if there is a packagegroupcategory with the same: id

Delete

Delete a single package group category

Status: complete

Path: /repositories/<repoid>/delete_packagegroupcategory/

Method: POST

Parameters:

  • categoryid: package group category id

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such package

Status: complete

Path: /repositories/<repoid>/add_packagegroup_to_category/

Method: POST

Parameters:

  • categoryid: package group category id
  • groupid: package group id

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such repository
