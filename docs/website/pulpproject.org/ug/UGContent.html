Pulp

Content

A content corresponds to any file type including RPM, SRPM, isos, msi etc in the Pulp server, regardless of whether it was synchronized from a repository feed or manually uploaded by a user.

Upload

Uploads content to the pulp server and stores them in a central location. If a repository is specified the uploaded content will be associated to the repo. The following arguments are optional:

Name Flag Description
RPM Directory --dir Local path to the directory containing the files to upload.
REPO ID --repoid Identifies the repository into which the content will be uploaded. specify multiple to associate to more than one repo.
NOSIG --nosig pushes unsigned content(rpms)
CHUNKSIZE --chunksize chunk size to use for uploads. Default:10485760
VERBOSE --verbose provides a more verbose output.

Example: Upload a directory of content to the Pulp server and associate it to a repository

$ pulp-admin content upload --dir /root/cloud-rpms --repoid testrepo

Example: Upload content to the Pulp server and associate them to a repository

$ pulp-admin content upload --repoid cloud zbar-qt-devel-0.10-6.fc14.i686.rpm

Example: Upload content to the Pulp server

$ pulp-admin content upload zbar-qt-devel-0.10-6.fc14.i686.rpm

Example: Upload an iso to the pulp server

$ pulp-admin content upload fedora-14.iso

Delete

Delete uploaded content from pulp server. The delete will skip if content has any repo associations

Name Flag Description
FILES --filenames content filename to delete from server.
CSV --csv a content csv with list of files to remove;format:filename,checksum.

Example: Delete the content

$ pulp-admin content delete -f zbar-qt-devel-0.10-6.fc14.i686.rpm

Example: Delete the content using a csv

$ pulp-admin content delete --csv=package.csv

List

List available content on pulp server. This should list any content type with format filename,checksum

Name Flag Description
ORPHANED --orphaned list of orphaned content.
REPOID --repoid list content in specified repo

Example: List the orphaned content

$ pulp-admin content list --orphaned

Example: List the repo content

$ pulp-admin content list --repoid=testrepo
