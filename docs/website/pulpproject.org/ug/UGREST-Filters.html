Filters

Repo filters are ways of filtering available source content for repositories.

Base URL Path: /filters/

Create

Create a filter

Status: COMPLETE

Path: /filters/

Method: POST

Parameters:

  • id: required filter id
  • type: required filter type - blacklist/whitelist
  • description: optional description for the filter
  • package_list: optional packages to be filtered - name or full NEVRA or python re syntax compatible regexes.

Authentication:

username/pass

Response: 201 Created

Headers: 

{'id': 'filter1',
  'description': 'filter1',
  'type': 'blacklist',
  'package_list': 'grinder'}

Error: 409 Conflict if there is an existing filter with the same id
