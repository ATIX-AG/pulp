Once you get a tepository created you can start managing package groups. Package groups exist as members of a repository.

Base URL Path: /repositories/<repoid>/packagegroups/

List

List the package groups under a repository

Status: complete

Path: /repositories/<repoid>/packagegroups/

Method: GET

Parameters:

None

Authentication:

username/pass or Consumer Certificate

Filters:

None

200 OK 

{
u'demo': {
  u'mandatory_package_names': [], 
  u'description': u'', 
  u'repo_defined': False, 
  u'default': True, 
  u'display_order': 1024, 
  u'immutable': False, 
  u'user_visible': True, 
  u'translated_name': {}, 
  u'translated_description': {}, 
  u'conditional_package_names': {}, 
  u'default_package_names': [u'zsh', u'zsh-html'], 
  u'optional_package_names': [], 
  u'langonly': None, 
  u'_id': u'demo', 
  u'id': u'demo', 
  u'name': u'demo_name'}
}

Error: 404 Not Found if there is no repository

Create

Create a new package group

Status: partial - location header not properly filled in

Path: /repositories/<repoid>/create_packagegroup/

Method: PUT

Parameters:

  • groupid: package group id
  • groupname: package group name
  • description: package group description

Authentication:

username/pass

Filters:

None

Response: 201 Created

Error: 409 Conflict if there is a packagegroup with the same: id

Delete

Delete a single packagegroup

Status: complete

Path: /repositories/<repoid>/delete_packagegroup/

Method: POST

Parameters:

  • groupid: package group id

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such package

Add Package

Add a package to a package group

Status: complete

Path: /repositories/<repoid>/add_package_to_group/

Method: POST

Parameters:

  • groupid: package group id
  • name: package name
  • type: package list type, example: {mandatory, default, optional}

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such repository

Delete Package

Delete a package from a package group

Status: complete

Path: /repositories/<repoid>/delete_package_from_group/

Method: POST

Parameters:

  • groupid: package group id
  • name: package name

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such repository

Import a comps.xml and create groups and categories

Path: /repositories/<repoid>/import_comps/

Parameters:

  • comps: comps xml data stream

Authentication:

username/pass

Filters:

None

Response: 

True
