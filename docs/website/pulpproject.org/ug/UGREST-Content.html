Pulp exposes a content API to perform file specific operations.

Eventually all API methods with have a specification on which level of access is required to call them.

Base URL Path: /content/

Lookup File

lookup files on pulp server by id.

Status: complete

Path: /content/files/

Method: GET

Parameters:

{'id' : fileid}

Authentication:

username/pass

Filters:

None

Response: 200 OK

Delete a File

Delete a single file

Status: complete

Path: /content/files/<file id>/

Method: DELETE

Parameters:

None

Authentication:

username/pass

Filters:

None

Response: 200 OK 

True

Error: 404 Not Found if there is no such file
